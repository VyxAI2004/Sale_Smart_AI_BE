# FastAPI To-Do App Backend

This project uses FastAPI, PostgreSQL, Poetry, and Alembic.

## Project Structure
- models/
- schemas/
- repositories/
- services/
- controllers/
- core/
- migration/
- shared/
- app.py
- env.py

## Setup
1. Install dependencies with Poetry
2. Configure PostgreSQL connection in `env.py`
3. Run Alembic migrations
4. Start FastAPI app
